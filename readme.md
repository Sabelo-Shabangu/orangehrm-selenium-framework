# OrangeHRM Selenium Automation Framework

## 📌 Project Overview
This is a simple QA Automation framework built using:
- Python
- Selenium WebDriver
- PyTest
- Jenkins CI/CD

It automates login testing on the OrangeHRM demo site.

Target site:
https://opensource-demo.orangehrmlive.com

---

## 🛠️ Tech Stack
- Python 3.10+
- Selenium 4+
- PyTest
- pytest-html
- Jenkins (CI/CD)
